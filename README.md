# 🌟 SocialFeed

A full-stack social media feed application built with React.js, Node.js, Express, and MongoDB.

## 🔗 Live Demo
- Frontend: https://socialfeed-frontend.netlify.app
- Backend: https://socialfeed-backend.onrender.com

## ✨ Features
- 🔐 JWT Authentication (Signup / Login)
- 📝 Create posts with text and/or images
- ❤️ Like / Unlike posts instantly
- 💬 Comment on posts in real-time
- 🗑️ Delete your own posts
- 🖼️ Click image to expand fullscreen
- ❤️ Double-click to like (Instagram style)
- 🔔 Toast notifications for all actions
- 🔍 Search posts and users
- 📱 Fully responsive design

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React.js (Vite) + Material UI |
| Backend | Node.js + Express.js |
| Database | MongoDB Atlas |
| Auth | JWT + bcryptjs |

## 🚀 Setup Instructions

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
